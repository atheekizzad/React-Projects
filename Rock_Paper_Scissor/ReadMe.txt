# 🪨📄✂️ Rock Paper Scissors Game (React)

A fun, interactive Rock Paper Scissors game built with React!  
Includes animated UI, glowing buttons, audio feedback, result tracking, and a downloadable game history feature.

---

## 🚀 Features

- 🎮 Classic Rock-Paper-Scissors logic
- ✨ Stylish UI with animations and modern CSS
- 🔊 Sound effects for Win / Lose / Draw / Click
- 🧠 5-round match system with winner announcement
- 📜 Game history log (last 5 rounds)
- 📁 Export game history as `.txt` or `.json`
- 🔁 Play Again / Reset options

---

## 📜 Rules of the Game

1. You and the computer choose between:
   - 🪨 Rock
   - 📄 Paper
   - ✂️ Scissor
2. Outcomes:
   - Rock crushes Scissor
   - Paper covers Rock
   - Scissor cuts Paper
3. If both choices are the same → it's a **Draw**
4. The first to win more rounds out of 5 is declared the winner.

- React (Functional Components + Hooks)
- CSS (Custom design with animations and shadows)
