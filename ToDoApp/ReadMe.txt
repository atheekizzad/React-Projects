# ToDo App with React & Node.js (Express)

> A simple ToDo list application built with React for the frontend and Node.js with Express for the backend. Tasks are stored in a JSON file on the server.

## Features

- Add tasks with a name and scheduled datetime
- Mark tasks as completed or pending
- Edit tasks (name and schedule) while retaining completion status
- Delete tasks
- Tasks persist on the server using a JSON file
- Frontend built with React hooks and functional components
- Backend built with Node.js and Express
- Basic validation and user feedback
