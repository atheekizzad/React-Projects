# 🎨 Color Palette Generator

A React-based color palette generator that creates random colors, displays their **RGB**, **HEX**, and **name**, and lets you copy the details to your clipboard with a single click.  
Built with **React**, **chroma-js**, **color-namer**, and **react-toastify**.

---

## 🚀 Features

- ✅ Generate a customizable number of random colors
- ✅ View **color name**, **RGB**, and **HEX** values
- ✅ Hover to preview color details
- ✅ Copy color details with one click
- ✅ Toast notifications for actions
- ✅ Accessible & responsive design

---

## 🛠️ Tech Stack

- **React** (UI framework)
- **chroma-js** (RGB ↔ HEX conversion, contrast checking)
- **color-namer** (naming colors from HEX values)
- **react-toastify** (toast notifications)
